## 🚀 Project Overview

House prices are influenced by several factors such as living area, number of bedrooms, and bathrooms. This project leverages **Machine Learning (Linear Regression)** to estimate property prices based on these key features.

To enhance user experience and demonstrate frontend integration with Machine Learning concepts, the project was developed as **two visually distinct interactive dashboards**, each providing real-time predictions and data visualizations.

---

## 🎯 Problem Statement

Develop a Machine Learning model capable of predicting house prices using housing characteristics and present the results through an interactive web-based dashboard.

### Input Features

| Feature      | Description                            |
| ------------ | -------------------------------------- |
| GrLivArea    | Above-ground living area (square feet) |
| BedroomAbvGr | Number of bedrooms                     |
| FullBath     | Number of full bathrooms               |

### Target Variable

| Variable  | Description                   |
| --------- | ----------------------------- |
| SalePrice | Estimated house selling price |

# 📈 Model Performance

The Linear Regression model was evaluated using standard regression metrics:

| Metric                         | Result       |
| ------------------------------ | ------------ |
| Mean Absolute Error (MAE)      | $34,859      |
| Mean Squared Error (MSE)       | 2.28 Billion |
| Root Mean Squared Error (RMSE) | $47,698      |
| R² Score                       | 0.5812       |

### Key Insight

The model successfully explains approximately **58.12% of the variance in house prices**, demonstrating a solid baseline performance using only three major housing features.
