# 🌙 House Price Prediction Dashboard (Dark Theme)

## Overview

This project is a modern dark-themed version of the House Price Prediction Dashboard developed during my Machine Learning Internship at Prodigy InfoTech.

It uses a Linear Regression model to estimate house prices based on property features.

## Features

### 📊 Analytics Dashboard
- Evaluation Metrics
- Feature Coefficients
- Model Equation

### 📈 Interactive Charts
- Actual vs Predicted Prices
- Sale Price Distribution
- Correlation Heatmap
- Residual Error Analysis

### ⚡ Live Prediction System
Users can:
- Modify square footage
- Adjust bedrooms
- Adjust bathrooms
- Generate real-time predictions

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- Machine Learning Concepts

## Dataset

Kaggle House Prices Dataset
